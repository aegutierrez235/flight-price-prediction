# ✈️ Análisis Predictivo: Precios de Vuelos (Business vs Economy)

Este proyecto aplica técnicas de Machine Learning para predecir el costo de boletos de avión, manejando datasets con distribuciones de precios altamente divergentes.

## 📈 Aspectos Destacados
- **Automatización de ETL:** Creación de una función de limpieza que procesa datos de distintas clases de vuelo manteniendo la integridad.
- **Ingeniería de Variables:** Transformación de datos temporales (duración y horarios) para capturar la estacionalidad diaria.
- **Modelado:** Implementación de Random Forest Regressor, logrando una excelente segmentación entre los grupos de precios 'Business' y 'Economy'.

## 🛠️ Herramientas
